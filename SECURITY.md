## Reporting a Vulnerability

We take the security of this project seriously.

Please report any security vulnerabilities by creating a private vulnerability report on GitHub at [https://github.com/salvatorecorvaglia/salotto/security/advisories/new](https://github.com/salvatorecorvaglia/salotto/security/advisories/new) or by reaching out to the maintainers directly.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress.
